# Employee Management System (EMS)

A Full Stack Employee Management System built using Spring Boot, JWT Authentication, and HTML/CSS/JavaScript.

This system allows organizations to manage employee leaves, track daily productivity, and monitor work reports through separate Admin and Employee dashboards.

---

## 🚀 Features

### Authentication & Security
- JWT Based Login Authentication
- Role Based Access Control
- Separate dashboards for Admin and Employee
- Secure API access using tokens

---

### 👨‍💼 Employee Features

Employees can:

- Apply for leave
- View leave status (Approved / Rejected / Pending)
- Add daily productivity logs
- View all their work logs
- Filter productivity by date
- See total hours worked
- View profile information (Name & Role)

---

### 🧑‍💻 Admin Features

Admin can:

- View all employee leave requests
- Approve or reject leave applications
- View employee work summaries
- Track total work hours
- Manage leave records efficiently
- Navigate through large datasets using pagination

---
